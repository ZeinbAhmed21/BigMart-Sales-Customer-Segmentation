# 🛒 BigMart Sales Customer Segmentation

## 📌 Project Overview
This project focuses on customer segmentation using unsupervised machine learning techniques.  
The goal is to identify meaningful patterns and group similar sales behaviors together to generate business insights.

Using clustering algorithms, we segment the dataset into distinct groups that can support strategic decisions such as targeted marketing, inventory optimization, and revenue analysis.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing
- Apply feature engineering techniques
- Reduce dimensionality using PCA
- Implement multiple clustering algorithms
- Evaluate clustering performance
- Generate business insights from the clusters

---

## 📊 Dataset

**Dataset Name:** BigMart Sales Dataset  
The dataset contains sales-related features such as:

- Item characteristics
- Outlet information
- Sales values
- Store types
- Location types
